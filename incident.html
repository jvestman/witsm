Incident Request Form

You should use this form to report an incident when a service is not working according to specification.

What is an incident?
An incident is any event where a service is not performing as expected. This includes errors, service downtime, degraded performance, or unexpected behavior.

What is not an incident?
Requests for new functionality, changes, or standard support are service requests.
Use the Service Request Form for those.

Priority guidance:
- Low: Minor annoyance, does not impact work.
- Medium: Hindering work, but workaround exists.
- High: Blocks work, critical functionality affected.
- Critical: Major outage, affects many users or business-critical services.

Please provide as much detail as possible, including symptoms, error messages, affected services, and steps already taken.
